HNAS

Version 1.0.0

Home Network Attached Storage (HNAS) 是一個現代化的網路附加儲存解決方案， 採用 Python Flask 框架開發，提供直觀的 Windows 11 風格使用者介面。 讓您輕鬆管理家庭或小型辦公室的檔案儲存與共享需求。
